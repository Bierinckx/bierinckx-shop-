# 🛍️ Bierinckx Shop

> Een moderne webshop applicatie met geavanceerde e-commerce functionaliteiten

[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](https://choosealicense.com/licenses/mit/)
[![Node.js](https://img.shields.io/badge/Node.js-18+-green.svg)](https://nodejs.org/)
[![JavaScript](https://img.shields.io/badge/JavaScript-ES2023-yellow.svg)](https://developer.mozilla.org/en-US/docs/Web/JavaScript)

## ✨ Features

- 🛒 **Productcatalogus** - Uitgebreide productweergave met categorieën
- 🛍️ **Winkelwagen** - Dynamische winkelwagen functionaliteit
- 💳 **Veilige betalingen** - Geïntegreerde betalingsprocessing
- 👤 **Gebruikersaccounts** - Registratie en profiel beheer
- 📱 **Responsive design** - Optimaal op alle apparaten
- 🔐 **Beveiliging** - Moderne security practices

## 🚀 Quick Start

### Vereisten

- Node.js 18+ 
- npm 8+
- Git

### Installatie

```bash
# Clone repository
git clone https://github.com/Bierinckx/bierinckx-shop-.git
cd bierinckx-shop-

# Installeer dependencies
npm install

# Start development server
npm run dev
```

### Environment Setup

```bash
# Kopieer environment template
cp .env.example .env

# Configureer je environment variabelen
# DATABASE_URL=...
# STRIPE_SECRET_KEY=...
# JWT_SECRET=...
```

## 📚 Scripts

```bash
npm run dev          # Start development server
npm run build        # Build for production
npm run start        # Start production server
npm run test         # Run tests
npm run lint         # Check code quality
npm run format       # Format code with Prettier
```

## 🏗️ Technologie Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES2023)
- **Backend:** Node.js, Express.js
- **Database:** MongoDB / PostgreSQL
- **Payments:** Stripe API
- **Authentication:** JWT
- **Testing:** Jest
- **Code Quality:** ESLint + Prettier
